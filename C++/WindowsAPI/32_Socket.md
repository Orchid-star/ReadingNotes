# Socket编程原理

- **网络程序 = 客户端程序 + 服务端程序**

- **套接字（Socket） = 网络地址（IP）+ 端口号（port）**

- ***Socket***

- **IP地址转换**

- **字节转换**

- **两种套接字标准**

  **Berkeley Socket  ---->  Linux/UNIX**

  **Winsock   ---->  Windows**

- **两种套接字**

  - **流套接字**

    **SOCK_STREAM**

    **TCP/IP**

  - **数据报套接字**

    **SOCK_DGRAM**

    **UDP/IP**
